Project Title

# Automated SB Admin 2 Deployment with NGINX

Project Description

"Automates the setup of NGINX and deployment of SB Admin 2 dashboard template on Ubuntu using a simple Bash script."

Features

One-line script execution

Sets up NGINX

Deploys SB Admin 2 from GitHub

Easy to view by accessing the server IP in a browser


Requirements

Ubuntu server

SSH access

Basic Bash knowledge


Setup Instructions

Step-by-step guide for users:

1. Clone the repository:

wget https://github.com/username/nginx-sbadmin2-setup.git  
cd nginx-sbadmin2-setup


2. Run the script:

chmod +x server.sh  
./server.sh
